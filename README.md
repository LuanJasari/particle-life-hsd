# Particle Life Simulator

## Projektbeschreibung
Dieses Projekt implementiert eine "Particle Life"-Simulation in Python. Ziel ist es, emergentes Verhalten durch die Interaktion tausender Partikel zu simulieren. Das System basiert auf einfachen Regeln (Anziehung/Abstoßung), die komplexe, lebensähnliche Strukturen erzeugen.

## Features
* **Simulation:** Dynamisches Partikelsystem mit >2000 Partikeln.
* **Interaktionen:** Definierbar über eine Interaktionsmatrix (Attraction/Repulsion).
* **Vielfalt:** Mindestens 4 verschiedene Partikeltypen (Farben).
* **Performance:** Optimierte Berechnung mittels NumPy (und ggf. Numba).
* **Visualisierung:** Echtzeit-Darstellung (Vispy/Pygame).
